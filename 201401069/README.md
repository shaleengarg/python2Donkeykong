This is the ascii version Of the game DonkeyKong first released in 1994

It can be played in Linux terminal and mac terminal

To play this game type the following in terminal

for Mac:
$ python main.py

for Arch linux:
$ python2 main.py

for other Linux Distros:
$ python main.py
